# Atividade-de-react

---
Catalógos de Lugares — Protótipo de App 

Protótipo educativo em React Native + Expo para demonstrar exibição de atrações de turismo sustentável.

Como rodar (Windows PowerShell)

1. Instale o Node.js e o Expo CLI ou use `npx`.

2. No PowerShell, execute:

```
cd c:\atividade\SN\reaact
npm install
npx expo start
```

Depois abra no emulador ou no app Expo Go no seu celular.

Arquivos criados:

- `App.js` — tela principal com lógica e interação.
- `components/AttractionCard.js` — componente personalizado com props.
- `package.json`, `app.json` — configuração mínima do projeto.

Observações:

- O app é um protótipo informativo (sem criação de listas de tarefas).
